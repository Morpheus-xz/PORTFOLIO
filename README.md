# Vedansh Agarwal Portfolio

Personal portfolio website built with **React + Vite + TypeScript**, with premium interactive visuals and smooth section-based storytelling.

## Live Website

https://portthefolio.netlify.app/

## Tech Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Three.js / React Three Fiber / Drei / Rapier

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Resume Download

Place the resume file here for in-site download buttons:

`public/Vedansh_Agarwal_Resume.pdf`

## Deployment (Netlify / Vercel)

Build command:

`npm run build`

Publish directory:

`dist`
